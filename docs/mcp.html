Run the server

Start the local stdio server, then point your agent client at it.

Start
onyx mcp serve

JSON examples

Keep requests local. Onyx handles the remote hop.

onyx_exec
{
  "name": "onyx_exec",
  "arguments": {
    "target": "hetzner-dev",
    "command": ["git", "status"],
    "cwd": "/srv/app",
    "timeout_ms": 30000
  }
}
onyx_logs
{
  "name": "onyx_logs",
  "arguments": {
    "target": "gpu-box",
    "job_id": "job_xxx"
  }
}

Streaming

Add stream: true to receive incremental notifications/progress events as output arrives, instead of buffering until completion.

Streaming exec
{
  "name": "onyx_exec",
  "arguments": {
    "target": "hetzner-dev",
    "command": ["git", "status"],
    "stream": true
  }
}

Events forwarded as progress notifications: started, stdout, stderr, reconnecting, resumed, timeout, finished.

Install the local client

Onyx installs locally. The remote server is provisioned automatically over SSH.

Homebrew
brew install shervin9/onyx/onyx
Shell installer
curl -fsSL https://useonyx.dev/install.sh | sh
Connect
onyx user@server

Core capabilities

Built around shells, jobs, and transport that hold up when the link does not.

QUIC + SSH fallback Prefer QUIC for the fast path, then fall back to SSH when UDP is blocked.
Session persistence (tmux-backed) Keep interactive shells reattachable with scrollback instead of starting from scratch.
Resumable exec onyx exec keeps commands running after client disconnects and lets you attach later.
Port forwarding Forward local ports to remote services through the same Onyx transport path.
ProxyCommand Use onyx proxy %h %p as a drop-in transport for SSH-based tools.
JSON output (--json) Emit structured NDJSON events for CI pipelines, agents, and log consumers.

QUIC when available. SSH when it's not.

  • QUIC + SSH fallback Fast UDP transport by default. Falls back to SSH silently if UDP is blocked.
  • Session persistence Network drops, laptop sleeps, Wi-Fi switches — your session waits on the remote.
  • tmux-powered Full scrollback, copy mode, and pane support. GPU jobs keep running.
  • AI-friendly Built for long-running workflows where a dropped connection costs hours.

Get started

curl -sSL https://useonyx.dev/install.sh | bash
$ onyx user@server

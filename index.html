How it works.

01

Bootstrap once

Run onyx user@host. On first connect, Onyx builds and starts the server binary over SSH. Nothing to configure manually.

02

Connect over QUIC

Subsequent connects go over QUIC/UDP — fast handshakes, built-in multiplexing, transparent reconnect on network drop.

03

Fall back silently

UDP blocked by a cloud firewall? Onyx falls back to plain SSH automatically. No flags. No intervention required.
